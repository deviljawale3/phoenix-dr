## Usage
Run the tool to check if a file is a JPEG:
```bash
cargo run test.jpg

\## How to Use



1\. Download the tool

2\. Run it from command prompt

3\. Choose options from the menu



\## Features



\- Recovers JPEG, PDF, PNG files

\- Safe read-only operations

\- Easy to use menu
